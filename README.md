# Accessible Switch Device

This repository contains the design and development resources for an accessible physical switch device intended to enable assistive interaction through alternative body movements such as foot activation.

The project is part of an academic initiative focused on the development of accessible and low-cost technological solutions for assistive environments.

## Project Overview

The device is designed to function as a physical interaction interface that allows users with limited upper limb mobility to activate functions using alternative body parts, such as feet.

The design focuses on:

- accessibility
- ergonomic interaction
- low-cost fabrication
- replicability through digital fabrication technologies such as 3D printing

## Repository Contents

The repository contains different resources related to the design and development of the device:

### 3D Models

Three-dimensional models of the device are developed using **Blender** and stored in the repository to maintain version control of the design process.

Files include:

- source design files (.blend)
- exported models for prototyping (.stl, .obj)

### Documentation

The documentation includes:

- conceptual sketches
- interaction diagrams
- hardware design notes
- design evolution


## Version Control

All design files are managed using version control through GitHub in order to:

- track design iterations
- enable collaboration between project members
- maintain traceability of changes
- support future development and improvements

## Project Context

This repository is part of a broader project organization that includes additional repositories for related systems such as firmware tools and assistive devices.

The goal of this structure is to ensure modularity and long-term maintainability of the project.
